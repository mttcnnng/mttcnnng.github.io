Harkster

An AI-powered institutional research platform for identifying consensus, uncovering hidden trade ideas and highlighting market-moving events across existing research feeds.

Visit Harkster ↗

What I built

An institutional research platform with a real-time Harkster Kernel that processed research feeds and powered proprietary AI assistants for alpha discovery through the product or internal-system integrations.

How I built it

Developed the platform using C#, Vue, ASP.NET and .NET Orleans, including support for bespoke AI assistants and agents tailored to client workflows.

Architecture

Clean Architecture, distributed systems, actor-based architecture, RESTful APIs

Deliverables

Marketing website · single-page application · REST API server · Model Context Protocol (MCP) server · .NET Orleans silo

Deployment

Microsoft Azure

Outcome

Helped users turn existing research into timely consensus signals, hidden trade ideas and market-moving event awareness.

Technologies

C# · ASP.NET Core Web API · Razor Pages · Vue.js · Tailwind CSS · Azure Cosmos DB · Azure SQL Database · Azure SignalR Service · Azure Storage · Microsoft Entra ID · Microsoft Graph · .NET Orleans · Azure Front Door · T-SQL
