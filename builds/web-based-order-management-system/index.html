Web-Based Order Management System

A multi-asset, SaaS-based order management system delivered through a web platform with a native iOS application.

What I built

An OMS with allocation, charge and FIX engines, plus workflow and trading features for multi-asset order management.

My contribution

Led development of the web platform, native iOS companion experience and backend architecture using C#, ASP.NET and SQL Server with event sourcing and CQRS.

Outcome

Supported scalability, resilience and complete audit history for complex order-management workflows.

Tech

C# · ASP.NET · SQL Server · FIX · Event sourcing · CQRS · Native iOS
