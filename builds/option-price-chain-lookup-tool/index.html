Option Price Chain Lookup Tool

A C# .NET Windows Forms application backed by Reuters market data for detailed option-chain lookup and analysis.

What I built

An option-chain tool showing expiries, strikes, bid and ask prices, last-traded values, volume, open interest, implied volatility and option Greeks.

How I built it

Developed the desktop application with filtering and comparison features for analysing liquidity, pricing and potential trading opportunities.

Architecture

N-tier architecture

Deliverables

Windows desktop application

Deployment

On-premises user desktops

Outcome

Gave traders a detailed view of complete option chains across expiries and strike prices.

Technologies

C# · .NET Windows Forms · custom Windows Forms controls · Oracle Database · SQLite · Reuters market data · ADO.NET
